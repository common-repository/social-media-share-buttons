Easy to use and 100% FREE social sharing plugin to add social media icons on your website without any Javascript & jQuery.

== Description ==

No.1 Fast Loading Social Media Sharing Plugin, build with simple coding. This plugin does not use any Javascript and jQuery.

Mostly, social sharing plugin uses Javascript for share icons and slow down your site, so it's time to ignore them and improve your site speed.

== Social Sharing Plugin Features==

This free social sharing plugin includes following features:

- No.1 fastest social share buttons plugin.
- This plugin does not affect your site speed.
- Add share buttons after the end of post/page.
- Add floating share button in left side of page/post.
- Pick your favorite social media share icons.
- Activate only required icons, leave disable others.
- Add Twitter Username and change it any time.
- Add **WhatsApp** share button that working on all device.
- Add **nofollow tag** or ignore this attribute (SEO).
- Social icons design worked on all widh-layout.
- Choose floating top paddings for more options.
- Hide social icons on mobile or set viewport size.

There is no need to load javascripts on every page. This plugin created for fast loading website that don't want use heavy loading plugins.

== Installation ==

= Installing this plugin - Simple =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, and search for **Social Media Share Buttons** and click "*Install now*"
2. Otherwise, <a href="https://downloads.wordpress.org/plugin/social-media-share-buttons.zip" target="_blank">download</a> plugin and upload to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate plugin, All Done!.
